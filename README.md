# Yen-Neram

Static-bundle multi-game shell on GitHub Pages. First game: 5-in-a-row.

**Status**: Pre-alpha. v1 in progress per [TODO/2026-06-07-v1-shell-and-5-in-a-row.md](TODO/2026-06-07-v1-shell-and-5-in-a-row.md).

## Quickstart

```bash
pnpm install
pnpm dev
pnpm test
pnpm build
```
